# 💧 AquaVision — AI Water Quality Monitor

AquaVision is a final year major project — an AI-powered water quality monitoring and decision-support system.

## Features
- 📊 Live Dashboard with real-time simulated sensor data
- 🤖 WQI Prediction using XGBoost / Random Forest / Linear Regression
- 🧠 SHAP Feature Importance Explainability
- 💡 Smart Rule-Based Recommendation System
- 📤 Bulk CSV Scanner with downloadable results
- 🗺️ Interactive Map with colour-coded quality markers
- 📧 Email Report Generation

## Tech Stack
- **Frontend**: HTML + CSS + Vanilla JavaScript + Chart.js + Leaflet.js
- **Backend**: Flask (Python)
- **ML**: scikit-learn, XGBoost, SHAP
- **Deployment**: Streamlit Cloud
